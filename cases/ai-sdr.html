Autonomous Lead Qualification

Built a 5-agent AI system that reads every incoming reply, classifies intent, qualifies leads through conversation, generates contextual responses, updates CRM, and schedules follow-ups — handling the entire SDR function across email and LinkedIn.

Role
Solo builder
Duration
6 weeks to production
Stack
n8n, OpenAI (5 agents), LangChain, Unipile, Close CRM, HeyReach
The Situation

The demand generation engine sends 100K+ emails/month and runs LinkedIn campaigns. Every reply needs classification (interested? bad fit?), qualification (team size? CRM?), a contextual response, CRM updates, and follow-up scheduling. One person doing this for that volume is the bottleneck. Hiring SDRs is expensive and slow. The question: can AI handle the entire function — not just draft replies, but qualify, route, and follow up autonomously?

The System

A 5-agent AI pipeline that processes every incoming reply end-to-end:

→

Agent 5 Send Format + deliver
≥ 3 users → Qualified (demo link)
< 3 users → Trial offer
2+ replies, unknown → Human
  1. Categorize
    Classifies each reply into 5 categories: interested, not interested, bad fit, later, or uncertain. Each category has specific rules ("thumbs up only" = not interested, "what's the pricing for 10 users?" = interested).
  2. Detect demo requests
    Checks if the lead explicitly wants a live demo or provided a referral contact. If yes → escalates to human via Slack with context. Not every interested lead needs a human — only demo-ready ones.
  3. Orchestrate
    Coordinates two sub-agents: a knowledge base agent that queries product docs for pricing/features/integrations, and an email creation agent that drafts the response using conversation context + KB answers.
  4. Qualify
    Tracks team size and CRM used across conversation turns. ≥3 users = qualified (demo booking link). <3 = trial offer. Unknown = weaves qualification questions into the response. If 2+ replies pass without getting team size → flags for human.
  5. Send + follow up
    Formats as clean HTML, replies to the original thread, logs to tracking sheet, schedules follow-up. Escalating cadence: 3 → 7 → 14 business days. CRM-aware — cancels follow-ups if lead status changed.
