Demand Generation Engine

Built a system that discovers high-converting audience segments, generates personalized outreach, and feeds conversion data back to sharpen targeting each cycle, generating $300K in pipeline.

The Situation

A B2B SaaS company with a $100/mo product needed enterprise-scale demand generation, with a lean team. At that price point, volume is everything: you can't grow by closing a few big deals. Every step was manual: finding audience segments meant eyeballing CRM data, writing copy took hours per segment, importing leads meant spreadsheet gymnastics. When a campaign ended, there was no systematic way to learn what worked. The team hit a throughput ceiling that couldn't scale without hiring.

The System

A 6-stage pipeline where each stage feeds the next, and results loop back to improve targeting:

Pipeline Flow
01 Enrich AI cleaning + ICP filtering
→
02 Import Import + CRM cleanup
→
03 Discover AI segment discovery
→
04 Write Copy from segment insights
→
05 Send Cold email + LinkedIn
→
06 Learn Self-building knowledge base
Feedback loop: conversion data refines next cycle's targeting
  1. Enrich
    Raw lead exports go through AI processing: normalization, country validation, title parsing, ICP filtering, deduplication. 400MB files in, clean targeting lists out.
  2. Import
    Every lead import doubles as a CRM cleaning session: deduplicates by URL → LinkedIn → email, flags conflicts, corrects stale data. Processing 100K leads/mo without failures.
  3. Discover
    AI analyzes converted customers to find why they closed: pain points from real conversations, industry patterns, company signals. Generates targeting criteria for each discovered segment.
  4. Write
    AI generates sequences per segment, grounded in pain points from real conversations, competitive research, and segment-specific value propositions. Language-aware (local language for single-country segments). Human approves before send.
  5. Send
    Cold email and LinkedIn campaigns execute with timezone-aware delivery.
  6. Learn
    Every conversion, reply, and meeting outcome feeds back into a knowledge base the system builds over time. Next cycle, segment discovery draws on this accumulated intelligence. The engine gets measurably sharper with each run.

Python, Claude Sonnet/Opus, Close CRM API, Apollo API, Smartlead API, HeyReach API, SQLite, GitHub Actions. Every stage uses idempotent state tracking: crash and restart produces zero duplicates. Segment discovery deduplicates across all previous runs so the system never re-targets the same audience. Human-in-the-loop approval gates before any campaign goes live.
